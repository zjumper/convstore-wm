# convstore-wm
Online shop sub-system for convestore.com site.

## Installation

### Prerequisites

* Install NodeJS/NPM
* Install bower

```shell
>npm install -g bower
>npm install -g grunt
```

* Install Git

### Download source code

```shell
>git clone https://github.com/zjumper/convstore-wm.git
```

### Build

```shell
>npm install
>bower install
```

## Run

```shell
>npm start
```
